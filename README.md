# Some cool Nintendo Games to emulate on your PC

Currently only contains Pokemon Games, might add to the collection

## Good practice:
Emulating a game will most likely create a .sav file, which the game needs for saving your progress. <br>
Because of this I would highly advise you to create a Folder somewhere (Desktop) labeled EMULATIONS or GAMES or whatever you want, and inside of that folder your gamefolders like pokemon-leaf-green-nuzlocke, after which you can put your .nds file (or whatever) inside. 

DO NOT play all of your games in the same folder. Will most likely cause problems later on.
